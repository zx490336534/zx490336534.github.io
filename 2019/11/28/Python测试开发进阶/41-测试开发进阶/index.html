  • 流编辑器,对文本进行逐行处理
  • sed [option] "pattern command" file
  • some command | sed [option] "pattern command"

常用选项[option]

-n只打印模式匹配的行

-f加载存放动作的文件

-r支持拓展正则

-i直接修改文件

匹配模式pattern

5 只处理第五行

5,10只处理第五行到第十行

/pattern1/只处理能匹配到pattern1的行

/pattern1/,/pattern2/只处理从匹配到pattern1的行到匹配pattern2的行
