参数化流水线

Jenkins pipeline中定义参数使用parameters指令，只允许放在pipeline块下

在http://127.0.0.1:8080/directive-generator/可以自动生成

  • defaultValue:默认值
  • description:描述信息
  • name:参数名 使用方法：${params.FLAG}

支持的参数类型

  • string

  • text
  • booleanParam
  • choice
  • file(有BUG不要用)
  • password
