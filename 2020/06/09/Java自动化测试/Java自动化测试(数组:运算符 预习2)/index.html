数组

数组是一种数据结构，它是一个容器，用来盛放数据。不同类型的数组可用盛放不同类型的数据。

特点

  1. 容量确定，定义的时候必须指定数组的大小。
  2. 类型确定，定义的时候必须声明数组的类型。

分类

  1. 一维数组
  2. 多维数组

创建一个数组

创建一个数组，但是没有初始化数组的元素。这种情况需要显式的指定数组大小。

数据类型[ ] 数组名 = new 数据类型[size];

创建一个数组，并已知数组的内部元素
