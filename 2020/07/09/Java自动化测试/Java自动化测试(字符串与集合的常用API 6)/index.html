字符串常用方法

equals

判断字符串是否一样，大小写敏感

在object类中

使用==会去判断引用地址是否相等，没有意义

在String类中进行了重写

equalsIgnoreCase

判断字符串是否一样，忽略大小写

一般用于验证码的验证

split

字符串切割

replace

替换字符串内容，不修改原先的内容，需要重新赋值

substring

截取字符串的一部分

该方法支持输入一个值：截取从该值索引到末尾

也支持输入两个值，截取从第一个值索引到第二个值的索引

trim

删除字符串两端空格

需要将所有空格删除可以使用replace

length

返回字符串长度

indexOf / lastIndexOf

  • indexOf 该值的第一个索引，如果不存在返回-1
  • lastIndexOf改值的最后一个索引，如果不存在返回-1

contains

判断内容是否在字符串中

isEmpty

判断是否为空

对null进行判断会报错

集合常用方法

除lang包外 其他包的调用都需要导包

add

泛型

控制集合的数据类型，只能是引用数据类型

get

返回一个内容

size

返回长度

remove

删除内容

删除不存在的内容返回false

set

为某索引的值设置新值
